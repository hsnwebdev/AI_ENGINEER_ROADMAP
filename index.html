Cheatsheet — Cost Function & Gradient Descent
▾
ConceptFormulaNotes
Cost function J(w,b)J = (1/2m) Σ (ŷ⁽ⁱ⁾ − y⁽ⁱ⁾)²Mean Squared Error / 2
Prediction ŷŷ = f(x) = wx + bModel output
Error per example(ŷ − y)²Squared to penalise large errors
Goalminimise J(w,b)Find best w, b
Gradient descent ww := w − α · ∂J/∂wUpdate simultaneously
Gradient descent bb := b − α · ∂J/∂bUpdate simultaneously
α (alpha)Learning rate (0 < α ≤ 1)Controls step size
J(w) graph shapeParabola (U-shape) vs wOne global minimum
J(w,b) graph shape3D bowl / contour plotMinimum at center of contours
